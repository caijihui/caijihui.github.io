关于Linux安装mysql查询显示出现乱码

发表于 2017-03-30 更新于 2024-08-27

原因分析

大概原因有以下几点：

1、网页head 没有设置 charset=”UTF-8”
2、mysql 数据库没有设置成uf8，set names utf8;
3、创建数据库和数据库字符没有设置utf8 ; default character utf8;
4、mysql 服务器和客户端没有设置成utf8

一些查看和修改步骤

一开始直接修改下数据库的编码：
mysql>alter database mydatabase character set utf8;
mysql>show variables like ‘character%’; #执行编码显示
